# Network Topology Project

## Latar Belakang
Project ini dibuat untuk memonitor jaringan menggunakan Zabbix.

## Tujuan
- Menampilkan status jaringan
- Monitoring device
- Analisa traffic

## Implementasi
- Menggunakan Zabbix
- SNMP monitoring
- Dashboard network

## Hasil
Monitoring jaringan menjadi lebih mudah dan terpusat.
